🔒 Confidentialité & RGPD

Données collectées

PulseUnit collecte uniquement les données nécessaires au fonctionnement de l'application : identifiant, prénom, rôle professionnel, code PIN chiffré, et les données de garde (assignations, checklist, planning).

Stockage des données

Les données sont stockées dans Firebase Firestore (Google Cloud, région Europe-West) et dans le localStorage de votre appareil. Les notes personnelles de lit sont exclusivement locales (jamais transmises).

Accès aux données

Seuls les utilisateurs enregistrés peuvent accéder aux données de garde. L'administrateur dispose d'un accès en lecture globale. Aucune donnée n'est partagée avec des tiers.

Durée de conservation

Les données de garde sont conservées le temps d'utilisation de l'application. Les notes de lit s'effacent automatiquement après 7 jours. Les réponses "travaillez-vous ?" expirent après 4 jours.

Sécurité

Les codes PIN sont stockés hachés (non réversibles). La connexion est sécurisée par HTTPS. Firebase applique des règles de sécurité limitant l'accès aux utilisateurs authentifiés.

Vos droits (RGPD art. 13-14)

Vous disposez d'un droit d'accès, de rectification et de suppression de vos données. Pour exercer ces droits, contactez l'administrateur de l'application. Aucune décision automatisée ni profilage n'est effectué.

Important

PulseUnit est un outil d'organisation interne. Ne saisissez jamais d'informations patient nominatives dans cette application.

🔑Connexion & Compte›
Première connexion — Crée ton compte : prénom · nom · rôle · code à 6 chiffres.
Ensuite — Recherche ton nom → sélectionne → tape ton code. Coche « Se souvenir » pour 7 jours d'auto-connexion sur cet appareil.
Mauvais nom ? Clique la ✕ à droite pour relancer la recherche.
Code oublié — « Code oublié ? » envoie une demande à l'admin qui génère un code provisoire (à remplacer dès la 1ère connexion).
3 codes faux = compte bloqué. Déblocage uniquement par l'admin.
Accès admin — Bouton rouge Accès Admin à la connexion. Identifiants distincts du compte personnel.
👥Rejoindre la garde›
À la connexion : OUI je suis de service → tu rejoins l'effectif. NON → mode lecture seule.
Les compteurs IDE / AS se mettent à jour en temps réel. Présence partagée jour ↔ nuit.
💡 Astuce — En garde de jour, la nuit à venir apparaît dans la barre des gardes : prépare l'équipe avant 20h.

📝Notes de lit›
Double-tap sur une carte de lit pour ouvrir la note.
5 emplacements indépendants par lit (Garde 1 à 5), conservés 7 jours puis effacés automatiquement.
Une note est privée : visible uniquement par son auteur. Réservée à l'IDE/AS assigné, à l'IDE Tech et aux médecins de garde.
L'horodatage de la dernière modification s'affiche en haut de chaque note.

saveData()Sauvegarde roster + shiftHistory en localStorage — alerte si quota dépassé
